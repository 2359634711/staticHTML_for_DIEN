# staticHTML_for_DIEN 大庆国际教育网静态页面




## 概述：
> 此项目为比赛项目，内容为大庆国际教育网静态`HTML`页面，主要分为3部分：`HTML`，`CSS`，`JS` 项目内容及动态效果采用`原生js`书写，轮播图使用轮播插件。
> 此项目基于`jquery`编写，其中涉及大量的基层DOM操作(虽然现在流行的前端框架大多数尽量避免DOM操作，但是对DOM基础的学习和理解是必不可少的)
